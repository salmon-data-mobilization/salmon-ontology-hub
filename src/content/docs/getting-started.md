## Prerequisites

- Node.js 18+
- npm (or your preferred package manager)

## Run the site locally

```bash
npm install
npm run dev
```

Visit `http://localhost:4321` while the dev server is running.

## Content basics

- Docs live in `src/content/docs/`.
- Add new pages as `.md` or `.mdx` files.
- Update `astro.config.mjs` to adjust the sidebar or site metadata.
